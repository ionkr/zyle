사용 방법

1. 우측 하단의 보라색 버튼을 클릭하여 분석 패널을 열 수 있습니다.
2. 버튼은 드래그하여 원하는 위치로 이동할 수 있습니다.
3. 위의 버튼들을 클릭하여 다양한 로그와 에러를 발생시켜 보세요.
4. 패널에서 로그를 클릭하면 상세 분석 결과를 확인할 수 있습니다.
